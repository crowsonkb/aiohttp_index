aiohttp\_index
==============

`aiohttp.web <http://aiohttp.readthedocs.org/en/stable/>`__ middleware to
serve index files (e.g. index.html) when static directories are requested.

Dependencies
------------

-  Python 3.5+
-  `aiohttp <http://aiohttp.readthedocs.org/en/stable/>`__ (tested on
   0.21.4)
